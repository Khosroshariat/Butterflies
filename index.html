The code :

First, I use the document.querySelector method to target and use the specific HTML element as a screen to move the mouse over it to make butterflies. Then I used the addEventListener function to make butterflies, in this function: I created two variables(xPos, yPos) to set the position of the butterflies on the screen, and to animate these butterflies I used the animation method(keyframes) inside the CSS file for the “h4” elements. I also used “document.createElement()” , “Math.random()” and “appendChild()” methodes to make the butterflies.
1. document.createElement() method: By using this method, I created “h4” element and stored it in a variable named “butterflies” to make the butterflies. But first I styled the “h4” element inside the CSS file using the different CSS properties to make sure the butterflies would appear on the screen correctly. (You can use any HTML tag element to make the butterflies but make sure you do not use this tag element inside the entire project for another thing.)
2. Math.random() method: By using this method, I create a random size for the butterflies each time they appear on the screen. Then by using the “style” method, I target the specific property in the CSS file for the “h4” element which are width and height in my project to replace them with these random sizes.
3. appendChild() method: By using this method, I added the butterflies that were created to the screen that I targeted at the beginning. In the end, I use the setTimeout() function to remove the butterflies from the screen after a specific amount of time.
